📒 Manipulador de Agenda em Python

📌 Sobre o Projeto

Este projeto consiste em um sistema de gerenciamento de contatos desenvolvido em Python, executado via linha de comando (terminal). O sistema permite ao usuário criar, visualizar, editar e excluir contatos, além de exportar e importar os dados da agenda.

A agenda é estruturada utilizando dicionários e listas, onde cada contato possui um nome e pode conter diferentes formas de contato, como telefone, e-mail e endereço.

O projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de programação em Python, incluindo manipulação de estruturas de dados, funções, leitura e escrita de arquivos e interação com o usuário.


✨ Funcionalidades

• ➕ Adicionar novos contatos à agenda

• 📞 Adicionar diferentes formas de contato (telefone, e-mail ou endereço)

• ✏️ Alterar o nome de um contato

• 🔄 Editar informações de contato existentes

• 👀 Visualizar um contato específico

• 📋 Visualizar toda a agenda formatada

• ❌ Excluir contatos da agenda

• 📄 Exportar agenda para arquivo .txt

• 🗂 Exportar agenda para arquivo .json

• 📥 Importar agenda a partir de um arquivo .json



🛠 Tecnologias Utilizadas

• Python 3

• Biblioteca padrão json (manipulação de dados)

• Biblioteca padrão os (verificação de arquivos)
